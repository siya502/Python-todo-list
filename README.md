# Python-todo-list
A simple command-line To-Do List application built using Python. Users can add tasks, view tasks, and manage their daily activities through a menu-driven interface.
## Overview
This is a simple command-line To-Do List application developed using Python.
The project allows users to:
- Add tasks
- View tasks
- Exit the application
## Features
- Menu-driven interface
- Task management using Python lists
- User input handling
- Simple and beginner-friendly project
## Technologies Used
- anaconda navigator
- jupyter notebook
